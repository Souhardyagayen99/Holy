Isn't this interesting! And remember, links can be used together with a lot of other tags.

1. ELEMENT - <img src="https://ninjasfiles.s3.amazonaws.com/0000000000001394.png" alt="coding ninjas logo">
LINK TO ADD - http://blog.codingninjas.in/

2. ELEMENT - <img src="https://ninjasfiles.s3.amazonaws.com/0000000000001395.png" alt="coding ninjas love coding">

3. ELEMENT - <p>blog.codingninjas.in</p>
